Getting Started

For edit html/css/js/php files use any text editor. For example, free editor Notepad++.

For installing this template, there are a few requirements you need on your web server in order to get full functionality:
- PHP 5.2+
- PHP Mail Support

Also in browser JavaScript must be enabled. If a user does not have JavaScript enabled in their browser, the site does not load and a message is displayed informing them of the problem.

To install this template on your web server, download the archive from Themeforest and unzip to the local disk on your computer. There will be two folders: "Documentation" folder, and "HTML" folder with the template files. Next step is to upload the template files to your server. Using an any FTP Client (for example, Filezilla), upload necessary contents of the "HTML" folder to your server directory.

This is not a Wordpress theme, so cannot be installed with Wordpress.

FAQ

Retina

For using adaptation to retina displays is necessary to create two image files: normal and high-resolution (with Apple's modifier @2x).

Example for logo:
1. logo.png (200x200px) - for simple displays
2. logo@2x.png (400x400px) - for retina displays
More information about retina.js

Newsletter Form

This template contains newsletter form. All you need is to replace your actual e-mail address and the subject. An e-mail will be sent at your e-mail adress with subject you choose every time when new user have subscribed. Open the newsletter.php in the root and replace all instances of "__YOUR_EMAIL__" with your email address and "__SUBJECT__" with desired email subject. 


$address = "__YOUR_EMAIL__";

$subject  = "__SUBJECT__";

These scripts need PHP support and may not work on a personal computer without running Apache server or similar server.

After changes are made, it's necessary to make sure the newsletter form works properly.

Background YouTube Video in Block

You can use any YouTube video for the background. Replace your YouTube video code in assets/js/init.js file. 


function initVideoBg() {
  if (!device.tablet() && !device.mobile()) {
    $('#video').YTPlayer({
        fitToBackground: true,
        videoId: '__YOUTUBE_VIDEO_CODE__',
        playerVars: {
          autoplay: 1,
          showinfo: 0,
          branding: 0
        }
    });
  } else {
      $('#video').addClass('video-mobile-screen');
  }
}

Counter in Crowdfunding Layout

Replace your the collected amount in assets/js/counter.js: 


$('#score-counter').counter({
  start: 0,
  end: 38500, // Amount
  duration: 7000 // Counter speed
});

And replace the remaining percentage in attribute data-percent in index-3.html: 


<div class="score-graph" data-percent="23%">
